# EduConnect Project

A comprehensive Education Management System featuring AI Recommendations and Blockchain Certificates.

## Modules

- **[Backend (Node.js/Express)](./backend_express/README.md)**: Core API and Database.
- **[Mobile (Flutter)](./mobile/README.md)**: Student and Teacher Mobile App.
- **[AI Service (Python)](./ai_service_python/README.md)**: Recommendation Engine.
- **[Blockchain (Hardhat)](./blockchain/README.md)**: Certificate Verification.

## Quick Start Overview

1. **Backend**: `cd backend_express` -> `npm install` -> `npm run seed` -> `npm run dev`
2. **AI Service**: `cd ai_service_python` -> `pip install -r requirements.txt` -> `python -m uvicorn main:app ...`
3. **Blockchain**: `cd blockchain` -> `npx hardhat node`
4. **Mobile**: `cd mobile` -> `flutter run -d chrome`

See individual `README.md` files in each directory for detailed setup instructions.
